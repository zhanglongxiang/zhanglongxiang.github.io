Java第7次课 实验

发表于 2017-04-12 |

实验1：

  • 定义一个类Father，里面有eat方法，该方法输出“我吃了一碗饭”。再定义一个类Son,继承Father类，重写eat方法，该方法首先调用父类的eat方法，然后输出”我吃了两碗饭”。
  • 定义测试类PolyTest进行测试。
  • 子类怎么调用父类的eat方法，去百度搜索java super。

实验2：

  • 假定根据学生的3门学位课程的分数决定其是否可以拿到学位，对于本科生，如果3门课程的平均分数超过60分即表示通过（这个要求需要在本科生的类中定义一个方法），而对于研究生，则需要平均分超过80分才能够通过（这个要求需要在研究生的类中定义一个方法）。
  • 根据上述要求，请完成以下Java类的设计：
    • 设计一个基类Student描述学生的共同特征。
    • 设计一个描述本科生的类Undergraduate，该类继承并扩展Student类。
    • 设计一个描述研究生的类Graduate，该类继承并扩展Student类。
    • 设计一个测试类StudentTest，分别创建本科生和研究生这两个类的对象，并输出相关信息。
