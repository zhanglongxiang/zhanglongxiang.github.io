使用Sublime Text

发表于 2017-03-03 |

导语

  • 第一次实验课写代码用的编辑器是Windows自带的记事本，太低端了，没有语法高亮，没有代码自动缩进，总之很不好用。

  • 我们这节实验课使用一种很好用的代码编辑器：Sublime Text 3

Sublime Text介绍

  • Sublime Text是一款具有代码高亮、语法提示、自动完成且反应快速的轻量级编辑器软件。
  • Sublime Text的下载可以到SublimeText官网下载，注意选择你的系统安装版本，下载后，安装到你的计算机上。

Sublime Text的常用快捷键

  • 使用Sublime Text要用它的快捷键。写代码时，双手在键盘和鼠标的来回切换，鼠标在菜单点来点去，这种感受太差了。所以要用快捷键，尽量抛弃鼠标。
  • Sublime Text几个最常用的快捷键，你用上几遍，就会发现快捷键的威力：
    • 建立新文件： Ctrl+N
    • 打开文件： Ctrl+O
    • 保存文件： Ctrl+S
    • 选择代码语言： Ctrl+Shift+P，在命令框中输入java
    • 字体变大： Ctrl++
    • 字体变小： Ctrl+-
  • 用上边这几个快捷键，敲出你自己的代码，把文件保存到计算机上。到此为止，你把使用流程走了一遍，已经会用Sublime Text3来写源代码了。
