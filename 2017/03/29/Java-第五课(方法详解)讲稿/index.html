坦克大战思路

  • 已经学习了定义属性和方法。这节课详细学习方法。
  • 游戏：坦克大战
  • 上面的坦克地图可以转换为二维数组，坦克大战实际上就是对二维数组的操作。

简单的开始

  • 需要两个类：Tank类和Game类。
  • 先从简单入手，只有一个游戏目标（一辆坦克）。
  • 1、游戏启动，创建一辆Tank并指定3个格子让这辆Tank在这个位置。
  • 2、开始游戏，提示玩家猜测，然后检查是否命中Tank的格子。如果是，则递增numOfHits变量(击中计数器)的值。
  • 3、游戏结束，3格都命中时游戏结果（当numOfHits的值递增到3），并告诉玩家他们花了多少次才干掉这个Tank。

Tank伪码

  • 伪码是介于Java语言与正常语言的一种语言，伪码大致上包括3部分：属性的声明，方法的声明和方法的逻辑。
  • 伪码最重要的部分是方法的逻辑，因为它定义将来会发生“什么事”。
  • 经过分析，我们认为Tank伪码如下：
    • 声明 一个int类型的数组，存储Tank在地图上用了哪几个格子:locationCells；
    • 声明 一个int类型的变量，存储击中目标的次数:numOfHits；
    • 声明 一个checkYourself()方法，检查用户是否击中，然后返回”hit”、”misss”或”kill”这三种结果之一；
    • 声明 一个setLocationCells()方法，设置每一次游戏Tank在地图上的位置；
  • 上面四个声明中，第三个，checkYourself()方法是最重要的，现在详细写它的伪码
