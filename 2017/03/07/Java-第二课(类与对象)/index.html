Java第2次课 类与对象

发表于 2017-03-07 |
  • 我们上个学期学习了C语言，这个学期学习JAVA，用它们都是写计算机程序，它们有什么不同？
  • 它们最直观的不同是：语法不同，这个不同，我们在这里不讨论。深远的不同是：同一件事情流程，用C语言和JAVA语言来描述，看待事物的角度不一样。
  • 我们用一个例子，分别用C语言和Java来实现。这个例子都能达到用计算机程序来解决实际问题的目的，但是有不同，我们来run一下，体会它们的不同。

面向对象，类和对象是什么关系呢？

  • 人以类聚，物以群分。一个人是一个具体的存在，是一个真实的对象。很多具有相同特征的人划分在了一起，成为一类人。
  • 大体的描述是，类是对相同对象进行统一性描述。想让类执行，这句话是错误的，怎么可能让一类人一起吃饭？只能让某一个人吃饭。所以，用类进行定义来描述外在客观世界，然后用类生成一个一个的具体对象来做它们具体的事情。
  • 对象有状态和行为两大部分组成，状态称为“属性”，行为称为“方法”。既然类是对对象的描述，类中当然要有这两部分。
