本次课概述

  • 类中除了方法还可以有属性（也叫变量），用属性来更好的刻画类。
  • 例如：有学生(Student)类,用它来描述学生，假设学生有上课、考试这两个动作，把这两个动作写成方法。学生要有姓名、年龄，以此来区别每个学生，定义属性来存储姓名和年龄。

变量与数据类型

  • 变量里面存放的是数据，数据自然有大有小，我们可以把变量看成是“容器”，我们不希望存储空间有浪费，所以希望大数据放进“大容器”，小数据放在“小容器”。
  • 所以有了数据类型（就象是容器规格）这个概念，Java有自己的数据类型，我们只有了解这些“容器规格”，才知道怎么安排组织数据。
  • Java的数据类型分为两大类：（1）原生数据类型，（2）引用类型。

Java数据类型之第一大类：原生数据类型

  • 很简单，Java的原生数据类型有8种，分别是：
  • 数值（带正负号）：byte 宽度：8位，short 宽度：16位，int 宽度：32位，long 宽度：64位
  • 浮点数：float 宽度：32位 ，double 宽度：64位
  • 布尔值：boolean：true或false
  • 字符：char 宽度：16位

变量命名要遵守的规范
  • 名字必须以字母、下划线或$符号开头，不能用数字开头；
  • 避开Java保留字，比如class、static、public、main等等，这些都是Java保留字，不要用它们做为变量的名字。

Java数据类型之第二大类：引用数据类型

  • 在Java中，一个类就是一种数据类型，称为引用数据类型。Student类就是引用数据类型。
  • 有多少种引用类型呢？无穷无尽，有多少个类就有多少个引用类型。
