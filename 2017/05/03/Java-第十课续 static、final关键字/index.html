静态方法

  • 上面的例子，Math类的sqrt()方法，它永远都执行相同的工作：计算某个数开平方的值。如果你生成了1000个Math的对象，并且都执行sqrt(4)方法，所得的值永远都是2.0，这样太浪费了。换句话说，这个方法会对参数执行操作，但这操作不受对象状态的影响。唯一能够改变sqrt()行为的只有所传入的参数。
  • Java是面向对象的，但是处于某种特殊情况下，有些实用方法不需要生成对象，因为它的状态与对象无关。
  • 不生成对象就能够执行的方法，我们称为静态方法，用static关键字来进行修饰。

  • 要生成对象，然后 对象名.普通方法

静态方法不能调用非静态的东西

  • 静态的方法是通过类的名称来调用的，所以静态的方法无法引用到该类的任何实例变量。因为静态方法是属于类的，而变量属于对象的。
