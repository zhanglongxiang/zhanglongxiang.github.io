Java期末复习上机练习

发表于 2017-06-08 | 分类于 Java教学 |

数组

  • 定义整型数组，求出其中的奇数个数和偶数个数。
  • 定义整型数组，求出数组元素的和、数组元素的最大值和最小值，并输出所求的结果。
  • 给出10个整数，然后任意查询一个数字是否存在该10个数字内。
  • 编程统计字符串数组中某个字符串元素出现的次数。
  • 编程对10个数从大到小降序排序并输出。

类、属性、方法

  • 设计一个User类，其中包括用户名name、口令password等属性；体现封装性。有参和无参构造方法。方法void showInfo() 显示属性信息。编写测试类UserTest进行测试。
  • Table类有名称name，颜色color，高度height等私有属性；有参和无参构造方法；各属性的setter 和getter方法；public void display():显示各个属性。编写测试类TableTest进行测试。
  • 编写一个类 MyCircle，radius:代表圆半径的双精度数；有参和无参构造方法；属性的setter和getter方法；pubic double calPerimeter(): 计算当前圆对象的周长并返回（提示：使用Math.PI），
    pubic double calArea(): 计算当前圆对象的面积并返回。编写测试类MyCircleTest进行测试。

继承、抽象类、接口

  • 编写程序，并满足如下要求：

    1. 编写一个Car类，私有属性：品牌(mark)；有参和无参构造方法；方法public void drive( )；
    2. 定义Car类的子类SubCar，具有：私有属性：价格（price）-float型、速度（speed）-int型；
      无参和有参构造方法；public void display()方法：输出属性参数
    3. 定义测试类CarTest进行测试。

  • 编写程序，并满足如下要求：

    1. 学生类Student包含：成员变量姓名name，年龄age 和学位degree；无参构造方法和有参构造方法；相关属性的setter和getter方法；输出属性值的方法public void show()；
    2. 由Student类派生出本科生类Undergraduate。包含私有属性：专业specialty；无参和有参构造方法；构造方法要体现super和this的应用。输出属性值的方法public void show()；
    3. 编写测试类StudentTest测试。

  • 编写程序，并满足如下要求：

    1. 创建一个Vehicle类并将它声明为抽象类;属性：品牌(mark)；体现封装性；抽象方NumOfWheels()返回值类型是int型，显示车辆轮子数。
    2. 创建两个类Car和Bike继承Vehicle类：在Car类中，包含私有属性类型(type)和产地(Orgin)；无参和有参构造方法；属性的setter和getter方法；重写的方法；在Bike类中，包含私有属性颜色(color)；无参和有参构造方法；属性的setter和getter方法；重写的方法；
    3. 编写测试类VehicleTest进行测试。

  • 编写程序，并满足如下要求：

    1. Shape接口中声明了常量PI及抽象方法 double getArea()；
    2. Circle类和Rect类实现接口Shape：Circle类的私有属性double类型的radius；有参和无参构造方法；显示属性信息的void showInfo()方法；Square类的私有属性double类型的width和height；有参和无参构造方法；显示边长信息的void showInfo()方法；
    3. 编写测试类ShapeTest求圆和矩形的面积。

  • 编写程序，并满足如下要求：

    1. 定义抽象类Person；有属性姓名name、性别sex和年龄age；抽象方法void work()；体现封装性。
    2. 定义接口HuMan，方法void show()，显示属性信息。
    3. 定义一个Student类继承Person类，同时实现HuMan接口：属性有：学号id、学校school等私有属性；方法有：各属性的setter和getter方法；无参构造方法和带参数的构造方法；
    4. 编写测试类TestStudent 进行测试。

Swing

  • 创建一个有关闭功能的带菜单的窗口程序；添加一个“操作”和“编辑”菜单，在“操作”下添加两个菜单项：“打开”、 “关闭”。“打开”和“关闭”两项之间用分割线分开，“关闭”菜单项具有关闭程序功能。“编辑”菜单有两个菜单项：“复制”、 “粘贴”。
  • 编写JFrame窗口程序，包含一个按钮组件，点击按钮时应使窗口的背景色显示为黄色。
